Pengaturan

Pengaturan Aplikasi
Manajemen Data

Reset Data

Hapus semua data surat masuk dan keluar. Tindakan ini tidak dapat dibatalkan.

Bantuan

Panduan Penggunaan

  1. Buka menu Surat Masuk
  2. Isi formulir surat masuk dengan data yang valid
  3. Klik tombol Simpan untuk menyimpan data
  4. Data surat masuk akan muncul dalam daftar surat masuk

  1. Buka menu Surat Keluar
  2. Isi formulir surat keluar dengan data yang valid
  3. Klik tombol Simpan untuk menyimpan data
  4. Data surat keluar akan muncul dalam daftar surat keluar

  1. Buka menu Laporan
  2. Gunakan filter untuk memilih jenis laporan yang diinginkan
  3. Tentukan periode tanggal jika diperlukan
  4. Klik tombol Terapkan Filter untuk menampilkan data
  5. Gunakan tombol Ekspor atau Cetak untuk mengunduh atau mencetak laporan

Pada menu Pengaturan, Anda dapat:

  • Mengubah informasi instansi
  • Mengganti kode keamanan aplikasi
  • Membuat backup data
  • Mereset semua data aplikasi
Informasi Aplikasi
Tentang SISMAT

SISMAT (Sistem Manajemen Surat) adalah aplikasi berbasis web untuk mengelola surat masuk dan surat keluar secara digital.

Versi: 3.0.0

Dibuat dengan: HTML, CSS, JavaScript, Bootstrap 5
