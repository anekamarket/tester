Bantuan & Panduan

Panduan Penggunaan

Untuk menambah surat masuk atau keluar:

  1. Pilih menu Surat Masuk atau Surat Keluar
  2. Klik tombol Tambah Surat
  3. Isi form yang disediakan
  4. Klik tombol Simpan

Anda dapat mencari dan memfilter surat dengan:

  • Menggunakan kotak pencarian di bagian atas tabel
  • Memfilter berdasarkan tanggal dengan datepicker
  • Mengurutkan data dengan dropdown sort

Untuk melakukan backup data:

  1. Pergi ke menu Pengaturan
  2. Klik tombol Backup Data
  3. File backup akan diunduh secara otomatis

Untuk restore data:

  1. Pergi ke menu Pengaturan
  2. Klik tombol Restore Data
  3. Pilih file backup yang sebelumnya diunduh
