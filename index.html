Bantuan

Panduan Penggunaan

  1. Pilih menu Surat Masuk
  2. Isi semua field yang wajib diisi (ditandai dengan asterisk *)
  3. Klik tombol Simpan Surat Masuk untuk menyimpan data
  4. Data akan tersimpan dan muncul di tabel daftar surat masuk

  1. Pilih menu Surat Keluar
  2. Isi semua field yang wajib diisi (ditandai dengan asterisk *)
  3. Klik tombol Simpan Surat Keluar untuk menyimpan data
  4. Data akan tersimpan dan muncul di tabel daftar surat keluar

  1. Pilih menu Laporan
  2. Tentukan periode laporan
  3. Pilih jenis laporan (Surat Masuk, Surat Keluar, atau Semua Surat)
  4. Klik tombol Generate Laporan untuk menampilkan data
  5. Gunakan tombol Export Excel, Export PDF, atau Cetak Laporan untuk mencetak
