Bantuan dan Panduan

Panduan Penggunaan

  1. Buka menu Input Data
  2. Isi semua field yang wajib diisi (ditandai dengan *)
  3. Untuk jenis bantuan "Lainnya", isikan jenis bantuan pada field yang muncul
  4. Klik tombol Generate untuk membuat kode validasi
  5. Klik Simpan Data untuk menyimpan

  1. Buka menu Lihat Data
  2. Gunakan filter untuk menyaring data berdasarkan desa, jenis bantuan, atau tanggal
  3. Gunakan kotak pencarian untuk mencari data berdasarkan nama atau alamat
  4. Gunakan navigasi halaman untuk melihat data yang lebih banyak

  1. Buka menu Ekspor Data
  2. Pilih format ekspor (PDF, Excel, atau Cetak)
  3. Tentukan rentang tanggal jika diperlukan
  4. Klik tombol ekspor untuk mengunduh atau mencetak
Pemecahan Masalah

Pastikan semua field wajib telah diisi dan kode validasi telah dimasukkan. Periksa juga koneksi internet jika menggunakan penyimpanan cloud.

Pastikan browser mendukung fungsi cetak dan tidak ada pop-up blocker yang aktif. Gunakan browser terbaru untuk hasil terbaik.
